# Audio Assets

WAV files for praise music:

1. **mozart_eine_kleine.wav** - Mozart "Eine kleine Nachtmusik" K.525 opening phrase (~7s)
2. **bach_invention13.wav** - Bach Invention 13 in A minor BWV 784 opening phrase (~7.5s)

Both compositions are public domain. Generated from MIDI using FluidSynth + FluidR3 GM SoundFont.
